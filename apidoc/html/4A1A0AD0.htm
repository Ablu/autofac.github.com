Remarks

Generally speaking, developers will not directly interface with this method. The primary entry point to registering Enterprise Library dependencies is RegisterEnterpriseLibrary. However, if you have a specific Enterprise Library registration definition that you need to add to an Autofac container, this method is how you do it.

Examples

In this example, an Enterprise Library type registration is created manually and added to an Autofac container. In normal circumstances, you would get the Enterprise Library type registration from a configuration source.

C#
Copy
// Set up the EntLib type registration.
var registration = new TypeRegistration<MyService>(() => new MyService());
registration.IsDefault = true;
registration.Lifetime = TypeRegistrationLifetime.Singleton;

// Use the extension method to add it to the
// Autofac container.
var builder = new ContainerBuilder();
builder.RegisterTypeRegistration(registration);
var container = builder.Build();
